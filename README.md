# Histoire AI v1.0.0

_I am So Proud To Present You, A Convenient Chatbot That Can Launch Troubleshooters, Browse The Web , Play Songs and Much More_

## Features Overview
> ***Play Songs***
> ***Tell The Time***
> ***Browse The Web***
> ***Custom Commands***
> ***Customizable Themes***
> ***System Info***
> ***Voice And Text Interaction***
